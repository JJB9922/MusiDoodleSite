v0.0.2

Minimum barely-viable product. Still at a point where you might as well use notepad.

- Picking a chord now creates a drag-and-drop box that can be moved around.
- The colour pallette for all windows is now completely different.
- Technical: Gtest has been added, it runs on basic stuff but I haven't got it doing stuff with Qt Widgets yet.

v0.0.3 will exist when the drag and drop is fully functional, in that users will be able to create a chord component and use it as intended. At this point, bar save/load, it will be a usable & functional notebook for simple Chord/Lyric songs.

v0.0.1

v0.0.1 is the first version of MusiDoodle. It is a prototype, and is not intended for use in production. It is possible to put something together in it, but it is not recommended to use it for anything important. You cannot save or load tracks. You cannot export tracks. You cannot share tracks. You cannot collaborate on tracks. You cannot even change the name of a track. You can only create a track, and then edit it. You could take a screenshot. At the moment, it is probably about as useful as a piece of paper and a pencil. But it is a start.

Although, it does show you some chords, and allows you to pick them out. However, there is no hand-holding or interactive music theory functionality yet.

I will not make an executable for v0.0.1, if you have found this post somehow, you can go to the GitHub repository and build it yourself.

0.1.0 (beta) will exist when I can save and load tracks.
